STORY 9: Configuración de GIT en Entornos de desarrollo

12. Instala la extensión de git en tu entorno de desarrollo.

Desde visual studio code. Pulsamos en el icono de extensión en la barra lateral izquierda, abajo del todo, y en el buscador ponemos GitHub Repositories, seleccionamos y clicamos en instalar. Ya lo tenemos instalado.

13. Inicia sesión con la cuenta de git creada

Te diriges a la página git, clicas en Sign in, introduces usuario, contraseña y entrar.

14. Descarga el repositorio creado de tu proyecto EQUIPO1

En Visual Studio Code, una vez instalado GitHub Repositories, te diriges a Source Control en la barra izquierda.

En el apartado Start, haces clic en Clone Git Repository… Aparecerá un campo arriba en el que debemos introducir la url del repositorio que nos queremos descargar.

Una vez git localiza el repositorio mediante la url, nos solicita una carpeta donde bajar el repositorio.

Seleccionada la carpeta, se baja el repositorio y nos preguntará si queremos abrirlo en Visual Studio Code para empezar a trabajar con él. Y ya lo tenemos.
